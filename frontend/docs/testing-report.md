### 2. 发现的问题及修复

#### ❌ 问题 1：AppNavbar 缺少 computed 导入

**严重程度：** 🔴 高  
**发现时间：** 测试阶段  
**影响范围：** AppNavbar 组件无法正常工作

**问题描述：**
```typescript
// 错误的代码
import { ref, onMounted, onUnmounted } from 'vue'

const isAdmin = computed(() => {  // ❌ computed 未导入
  return authStore.user?.roles.includes('ROLE_ADMIN') || false
})
```

**修复方案：**
```typescript
// 修复后的代码
import { ref, computed, onMounted, onUnmounted } from 'vue'  // ✅ 添加 computed
```

**验证结果：** ✅ 已修复，热更新成功，无错误
