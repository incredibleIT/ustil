## 🐛 常见问题排查

### 问题 1：滚动渐入动画不触发

**可能原因：**
- Intersection Observer API 不被支持（旧浏览器）
- 元素初始样式未正确设置

**解决方案：**
```javascript
// 检查浏览器支持
if ('IntersectionObserver' in window) {
  console.log('✅ Intersection Observer 支持')
} else {
  console.log('❌ 不支持 Intersection Observer')
}
```

### 问题 2：页面过渡动画闪烁

**可能原因：**
- 过渡时长太短
- mode 设置不正确

**解决方案：**
```vue
<!-- 确保使用 out-in 模式 -->
<Transition name="page" mode="out-in">
  <component :is="Component" :key="route.path" />
</Transition>
```

### 问题 3：骨架屏不闪烁

**可能原因：**
- animated prop 设置为 false
- CSS 动画被覆盖

**解决方案：**
```vue
<!-- 确保 animated 为 true（默认值） -->
<SkeletonScreen type="card" :animated="true" />
```
