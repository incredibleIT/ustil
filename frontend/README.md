# frontend

## Environment Requirements

- **Node.js**: >= 20.20.2 (managed via `.nvmrc`)
- **Package Manager**: npm >= 10.0.0

**Note**: This project uses Vite 6.x which requires Node.js 14.18+. We recommend using [nvm](https://github.com/nvm-sh/nvm) to manage Node versions:

```sh
# Install nvm (if not already installed)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Switch to the correct Node version
cd frontend
nvm use  # Automatically reads .nvmrc file
```

This template should help get you started developing with Vue 3 in Vite.

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.
